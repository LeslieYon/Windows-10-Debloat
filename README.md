# Windows-10-Debloat
Windows 10 系统优化&amp;精简  
本项目致力于使用尽量少的第三方软件，实现对Windows 10系统的优化 ~~（Win7化）~~

注意：为保证效果，本项目中的注册表文件，请在OOBE阶段开始之前，使用Dism++工具的【系统优化->导入配置】功能导入。
